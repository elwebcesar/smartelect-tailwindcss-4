## Características

* [**TailwindCSS**](https://tailwindcss.com/") - 4.
* [**Vite**](https://vite.dev/) - para ejecución de TailwindCSS.
* [**Iconify**](https://iconify.design/) - biblioteca de íconos.


## Instalación

1. Clona el repositorio: `git clone https://github.com/tu-usuario/tu-repositorio.git`
2. Instala las dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`

## Comandos

* `npm run dev`: Inicia el servidor de desarrollo con Vite.
* `npm run build`: Compila el proyecto para producción con Vite.
* `npm run preview`: Previsualiza el proyecto compilado en producción.

## Dependencias

* **DevDependencies**:
